# Frontend

Frontend for the Task Manager, built with Vite + React + TypeScript.

## Setup
```bash
npm install
```

## Dev
```bash
npm run dev
```

## Tests
```bash
npm test
```

## Typecheck
```bash
npm run typecheck
```

## Lint
```bash
npm run lint
```

## UI foundation
Base UI components live in `src/shared/ui` and are backed by design tokens in
`src/app/globals.css`.
